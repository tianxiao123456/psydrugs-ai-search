# 维生素

维生素分为多种：维生素A、维生素B族、维生素C、维生素D、维生素E、维生素K等。遵医嘱或适量服用对身体有益。例如经典的维生素C，可以增强抵抗力、预防感冒等。

## 维生素A

维生素A是一种人类必需的是一组不饱和营养有机化合物，又被称为视黄醇（Retinol），是人类必须的营养素之一。它在维持人体正常视觉、免疫功能和皮肤健康方面起着至关重要的作用。

CAS号：68-26-8


### 维生素A的主要作用

- 保护视力
- 增强免疫力
- 促进组织修复
- 促生长

### 每日摄取上限^[[Wikipedia](https://zh.wikipedia.org/wiki/%E7%B6%AD%E7%94%9F%E7%B4%A0A)]

根据1mg = 1,000μg = 1,000微克
1国际单位 = 0.3微克维生素A = 0.332微克乙酰维生素A；

0-3岁：600微克
4-9岁：900微克
10-12岁：1,700微克
13-18岁：2,800微克
18岁以上：3,000微克

### 副作用
 
 摄入过多维生素A可能导致皮肤干燥、瘙痒、皮疹、头痛、恶心等症状。

### 注意事项

- 维生素A过量可能导致皮肤干燥、瘙痒、皮疹、头痛、恶心、发烧等症状。
- 孕妇和儿童应特别注意补充量，避免过量。

## 维生素B族

维生素B（B vitamins）又称维他命B、维生素B族（vitamin B family）、B族维生素，是一组水溶性维生素，对于细胞代谢及合成红细胞有着重要的作用[1]。虽然它们是一类化学性质多样的化合物^[[Wikipedia](https://zh.wikipedia.org/wiki/%E7%BB%B4%E7%94%9F%E7%B4%A0B)]
